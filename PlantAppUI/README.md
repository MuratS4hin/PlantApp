# PlantApp - React Native UI

A beautifully designed, modern mobile application concept for a Plant Care app. This project focuses on high-quality UI/UX components, smooth transitions, and a clean aesthetic built using **React Native**.

## ğŸš€ Getting Started

### Prerequisites

* Node.js (v14 or higher)
* npm or yarn
* Expo Go app on your mobile device (if using Expo) or an Android/iOS emulator.

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/MuratS4hin/PlantApp.git
cd PlantApp

```


2. **Install dependencies:**
```bash
npm install
# or
yarn install

```


3. **Start the application:**
```bash
npx expo start

```


4. **Run on a device:**
* Scan the QR code with your phone (Expo Go).
* Press `a` for Android emulator.
* Press `i` for iOS simulator.
